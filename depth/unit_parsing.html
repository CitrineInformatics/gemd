1.1. Unit ParsingÂ¶

Unit parsing is performed using the Pint package. By default, Pint supports a larger set of units than the Citrine Platform. Therefore, we include a custom unit definition file in GEMD-python: citrine_en.txt. This file contains the most commonly used units and will grow over time.

Requests for support of additional units can be made by opening an issue in the GEMD-python repository on github.

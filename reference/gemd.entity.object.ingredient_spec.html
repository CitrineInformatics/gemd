An ingredient specification.

Ingredients annotate a material with information about its usage in a process.

Parameters

  • uids (Map[str, str], optional) ‚Äì A collection of unique IDs.

  • tags (List[str], optional) ‚Äì Tags are hierarchical strings that store information about an entity. They can be used for filtering and discoverability.

  • notes (str, optional) ‚Äì Long-form notes about the ingredient spec.

  • material (MaterialSpec) ‚Äì Material that this ingredient is.

  • process (ProcessSpec) ‚Äì Process that this ingredient is used in.

  • mass_fraction (ContinuousValue, optional) ‚Äì The mass fraction of the ingredient in the process.

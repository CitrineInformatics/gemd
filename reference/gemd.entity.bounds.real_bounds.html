gemd.entity.bounds.real_bounds module¶

Bound a real number to be between two values.

class gemd.entity.bounds.real_bounds.RealBounds(lower_bound=None, upper_bound=None, default_units=None)¶

Bases: gemd.entity.bounds.base_bounds.BaseBounds

Bounded subset of the real numbers, parameterized by a lower and upper bound.

Parameters

  • lower_bound (float) – Lower endpoint.

  • upper_bound (float) – Upper endpoint.

  • default_units (str) – A string describing the units. Units must be present and they must be parseable by Pint. An empty string can be used for the units of a dimensionless quantity.

contains(bounds: gemd.entity.bounds.base_bounds.BaseBounds) → bool¶

Check if another bounds is a subset of this range.

The other bounds must also be a RealBounds and its lower and upper bound must both be within the range of this bounds object.

Parameters

bounds (BaseBounds) – Other bounds object to check.

Returns

True if the other bounds is contained by this bounds.

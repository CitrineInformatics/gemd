Module contents¶

Taurus JSON support, which provides a drop-in replacement for json.

This module provides the four main python json methods:

  • dump() for serializing python and taurus objects to a JSON file

  • load() for deserializing python and taurus objects from a JSON file

  • dumps() for serializing python and taurus objects into a String

  • loads() for deserializing python and taurus objects from a String

These methods should provide drop-in support for serialization and deserialization of taurus-containing data structures by replacing imports of json with those of taurus.json.

It also provides convenience imports of TaurusEncoder and TaurusJson. These classes can be used by developers to integrate taurus with other tools by extending the JSON support provided here to those tools.
